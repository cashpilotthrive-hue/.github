FBchat Examples v1.0

Overview

This repository provides practical examples for automating Facebook Messenger using the fbchat Python library.

  • Scheduled Messenger: Send messages at specific times daily.
  • Template Messenger: Use JSON-based templates for dynamic messaging.

Automation Features

The scripts include robust error handling for:

  • Authentication failures
  • Friend search misses
  • Invalid template selections
