Load Balancer Simulation Production

Overview

This is a high-performance simulation of a least-connections load balancer implemented in Python.

Key features include:

  • Min-heap based server selection ($O(1)$ selection)
  • Reduced lock contention for high throughput
  • Lazy logging to minimize I/O overhead
